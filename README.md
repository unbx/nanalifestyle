# NANA LIFESTYLE — Portfolio Site

Portfolio site for Sean Nana / NANA LIFESTYLE. Built with Next.js 14, Three.js (React Three Fiber), Tailwind CSS, and Framer Motion.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy to Vercel

```bash
npx vercel
```

Or connect this repo to Vercel for automatic deployments.

## Structure

- `src/app/` — Next.js App Router pages
- `src/components/` — React components (Hero, Creative, Product, Experience, Footer)
- `src/data/content.ts` — All site content as structured data
- `src/styles/globals.css` — Global styles, effects (film grain, scan lines, holo gradients)
- `public/assets/` — Brand assets (NANA logo SVG, images)

## Tech Stack

- **Next.js 14** (App Router)
- **React Three Fiber** + **Drei** (3D scenes)
- **Tailwind CSS** (utility-first styling)
- **Framer Motion** (scroll animations)
- **TypeScript**
